# 宜搭角色管理

在宜搭中具有相同功能人员的集合可以通过设置一个角色来实现。在宜搭中如下几个场景会使用到角色，流程审批人角色，流程发起角色，消息接受角色，模版打印角色，自定义页面浏览角色，报表浏览角色。例如：报销中，需要财务进行审批，如果有多个财务的情况下，可以设置一个财务的角色，添加对应的人员，在设置审批人时，直接选择财务这个角色，避免重复添加审批人，简化流程节点，节约时间。
## 角色和角色分组管理[​](#a1YS2)
使用管理员身份，登录[宜搭工作台](https://www.aliwork.com/workPlatform)。单击右上角**平台管理**![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1700653499495-cafaf794-410f-4099-ac18-6e134fc14278.png)
按钮，进入平台管理页面。![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1701051152649-8921a803-b739-437e-bc85-261c9f4cb23c.png)
在左侧菜单栏单击**角色管理**。![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1701051277219-664fbbc0-2645-4e79-975b-f5a931da9817.png)
选择**宜搭角色**，进行角色管理。支持按照角色组对角色进行管理。![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1748261519984-339febe2-a4ef-4bd8-8cf2-15e750f70746.png)
单角色管理：在**新增角色**对话框中，设置**角色名称**、**角色维护人**和**角色成员**，然后单击**确定**。
| **属性** | **说明** | **示意图** |
|---|---|---|
| **角色名称** | 需要新创建的角色的名称。名称全局唯一，不允许重名 |  |
| **分组到** | 角色放到指定分组下，便于管理。 |
| **角色维护人** | 角色维护人是负责管理该角色的人。角色维护人可以设置、修改和删除该角色中的成员。 |
| **角色成员** | 拥有该角色权限的成员。每个角色最多配置2000人 |
| **角色描述** | 请输入该角色的描述信息，最多300字符。 |
**补充：****角色创建权限**
只要拥有应用创建权限的用户，均可创建新角色。**角色可见性**
所有用户都可以看到平台中已经创建的角色（即角色创建完成后，全员可见该角色）。**角色的删除与修改权限**
角色的“删除”和“修改”等维护操作，仅限于该角色的维护人进行，其他用户无权操作。
钉钉角色只有宜搭平台管理员及钉钉主子管理员有权限管理和查看，其他人员不可见。钉钉角色在宜搭平台内仅展示，如果需要修改，需要登录到[钉钉管理后台](https://oa.dingtalk.com)中管理。在应用管理后台，选择角色的页面，可以同时选择到钉钉角色和宜搭角色。
角色分组管理支持单分组编辑，或角色分组批量编辑。单个角色分组编辑角色分组批量编辑![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1748261887623-e29e3837-542a-459b-99c8-258ff1de9670.png)

## 设置角色成员的管理范围[​](#pFj4X)
进入**角色管理**页面，然后选择**宜搭角色**。选择目标角色，然后在需要修改的角色成员后方单击**设置管理范围**。![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1748262099154-7a5f89fc-9283-482b-8093-37292a07980c.png)
在弹出的**选择人员**对话框内，选择当前人员可管理的权限范围，然后单击**确定**完成添加。![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1701072101438-6439eb1d-4730-455c-a588-8ead84b35de6.png)
角色设置完成后，可以添加到审批节点里，效果如下：![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/gif/1701136352853-d4e70fc1-042a-4073-9f19-de5ade2b95ed.gif)
