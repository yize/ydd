# 抄送节点

抄送人拥有所在节点的查看审批单权限，同时也支持设置抄送节点所能查看到的表单内容，可以实现一些简单的权限控制。
**说明**：
在审批单的流程链路中不会显示抄送节点。
**抄送节点使用场景**：当审批流程发起后，需要对应的人员查看/知晓该审批单，但是无需审批的场景。

设置抄送人所能看到的表单内容的路径如下：
点击对应的流程表单 > 流程 > 节点设置，如图：

主管审批完成后需要抄送发消息同步给指定的人员。
