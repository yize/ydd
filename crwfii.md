# 流程简介

本章目录如下：[流程简介](https://www.yuque.com/yida/support/crwfii)[快速开始](https://www.yuque.com/yida/support/qgaxvq)[流程节点](https://www.yuque.com/yida/support/trbqg6)[流程设置](https://www.yuque.com/yida/support/ef8e88)[流程分析](https://www.yuque.com/yida/support/vpdaq9)[代理中心](https://www.yuque.com/yida/support/tpnaqu)[流程任务中心](https://www.yuque.com/yida/support/hg13tb)[常见问题](https://www.yuque.com/yida/support/efsvv4)[流程案例](https://www.yuque.com/yida/support/bpnetv)**说明：**宜搭的简易流程设计器，最多至能支持100个设计节点（开始和结束节点也包含在内），超过会报错并且无法保存流程。宜搭专属版的流程设计器可支持 150 个节点，使业务流程编排更自由。
## 功能介绍[​](#ZRAou)
当表单需要人员审核确认时，就需要使用流程表单了。而流程表单和普通表单的最大区别，就是流程表单需要进行流程设计。
## 使用场景[​](#xp8OE)
由于传统的纸质审批费时又费力，特别在复杂业务流中，一份审批文件，发起人需要跑上跑下，审批人也要面临工作被打断的困扰；所以我们推出了宜搭，可以通过宜搭线上流程审批轻松的完成各种复杂业务流程的审批工作，促进团队工作效率的提升。
## 操作步骤[​](#NYu1S)
登录[宜搭工作台](https://www.aliwork.com/workPlatform)。选择目标应用，进入应用的**页面管理**页。选择需要编辑的**流程表单**，单击进入**表单设计**页。表单设计页，单击顶部**流程设计**。![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1726831555217-e4dcee52-23a4-4005-8f6e-5cc2a781ea02.png)
单击右上角**创建新流程**。**说明**：启用中的流程，仅支持查看，无法直接编辑，流程表单创建成功后会自动创建一个审批人为你自己，版本号为 V0 版本的流程。![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1726831749329-8a2414e7-29cb-4556-a49a-e8633b14b7e0.png)
## 常见问题[​](#inWr4)
**Q：新版简易流程设计器如何创建，为什么应用中的普通流程编辑器未更新为新版简易流程设计器？**目前新版的简易流程设计器适用于新建的流程页面中，更新前建立的流程仍为旧版流程设计器。**Q：宜搭 2.0 支持新版的简易流程设计器吗？**支持，页面可以点击蓝色框位置一键升级。升级后将不能再退回旧版流程设计器，已设置的流程会配置在新版中。![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1632989428671-b2a51ba2-fbfb-4341-acfb-d08bb5a7c2f2.png)
![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1632989450952-be24a8e0-7ee5-4db4-9bc1-98beb6fdb56b.png)
**Q：提交审批人的时候报错「审批人不合法，请联系企业管理员」？****出现该问题的原因**：是因为流程节点上设置的审批人不存在，因为宜搭简单流程在发起的时候会校验审批人是否符合流程设计。**解决方法：**一般都是设置审批人为主管的时候，会报这个错，那么去核实一下发起人在钉钉通讯录所在的部门里是否有设置对应的主管，如果没有，设置上主管就可以了还可以在流程节点上将审批人设置为「允许审批人为空」例如二级主管可能没有，此时需要允许二级主管为空，查看以下动图进行设置：![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/gif/1615806019346-5cf054f8-c534-49f4-90b0-3f310f772ff4.gif)
**Q：如何将审批结果通知发起人？**目前仅支持手机端审批人在审批的时候点击同意/拒绝按钮后，可以勾选底部通知发起人，电脑不支持。![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1637043654562-247a20cb-d05d-4a57-8de2-a3d96e8cf1ee.png)
勾选后进行提交，会在钉钉私聊会话框出现一条消息提醒，效果如下图：![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1637043705687-28e14db5-c777-4524-8301-7ce9a713b99c.png)
**Q：如何关闭移动端提交审批通知审批人？**移动端提交流程表单，默认会弹窗提示：该流程目前的处理人是：XX，是否通过聊天告知？（PC端不会提示）。若想要关闭此功能，可[**点击此处**](https://www.aliwork.com/o/closelcsprtz)填写表单提交申请 。表单提交后，宜搭会在三个工作日内处理。温馨提示：关闭此功能是根据组织维度关闭，请谨慎考虑。![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1645494037480-785166cf-4d83-4b80-88e2-1960a6116ada.png)
**Q：如何快速将流程表单转换成普通表单？**当前无法直接转换表单类型，可通过以下步骤实现：打开流程表单设计页面 >> Windows 按住 Ctrl 键/MAC 按住 Command 键 >> 选择表单设计上面的所有组件/点击左侧大纲树进行选择所有组件 >> 使用键盘快速复制按钮 >> 粘贴到新建的普通表单 >> 保存**注：此方法只会将表单配置快速复制到普通表单，若有配置 JS 或页面设置等内容，需要在新建的普通表单再进行单独设置。**![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/gif/1645601442751-08b56108-1dc2-406b-9d51-6a5334d652d2.gif)
**Q：流程发起后，为什么审批流程中少了一个审批人****出现该问题的原因可能是**：流程发起后，发起人的部门架构被调整，导致新的部门流程未生效。**Q：为什么审批任务生成后，没有收到工作通知？**A：从审批任务生成到审批工作通知发送之间，有一定的网络时间延迟，发送工作通知钱前会校验任务状态，对于已经完成的任务不再发送审批通知。例如：审批任务生成后，审批人立刻完成了该审批，但是工作通知可能还没来得及发送，此时该任务所对应的工作通知将不会发送。**Q：如何反馈流程分析的使用问题？**若在使用流程分析功能中遇到任何问题或建议反馈，可以扫码下方二维码申请加入群反馈。![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1643248110066-eae37666-9770-40f0-a378-2b4c6dbbdf67.png)
**Q：如何自定义流程分析的数据范围？**如果通过流程表单-->自建流程分析创建的默认报表，默认报表会默认本表的过滤条件，默认只查本流程。若在报表中手动新增组件并选择内置数据集，则需要自行在【筛选器过滤】中绑定的流程，否则默认是应用级别的全量明细数据**Q：设置了消息通知为什么没有生效？**流程表单若新增、修改、删除了消息模板，需要点击页面蓝色字体【点击此处】，将会自动创建新流程版本进行生效。![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1654827861264-ad482856-479e-40be-bc10-3bea1f97e8d6.png)

**Q：流程发起后，如果发起人离职或更改部门，此时流程会怎样运行？如果运行报错应该怎么处理？**流程发起人和发起部门在流程发起那一时刻已经确定了的，后续流程的全生命周期里均不会改变，无论是换部门还是代理给别人重新提交。如果流程中有人离职，且审批节点取人依赖于该离职人的组织架构信息，如主管审批等，则流程会出现找不到审批的错误，导致流程异常或无法继续审批，此时需管理员通过跳转节点解决。
