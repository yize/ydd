# 聚合方式

## 1. 简介[​](#PYQ4i)
将该字段下分散的数据聚集到一起统计，根据不同的字段类型，有不同的聚合方式可供选择
## 2. 操作步骤[​](#H1iLJ)
**操作路径：**报表页面 >> 表格列 >> 字段后面的编辑按钮 >> 数据设置面板 >> 聚合******字符串**：计数、计数（去重）**日期**：计数、计数（去重）、最大值、最小值**数值**：计数、计数（去重）、求和、平均值、最大值、最小值![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1682217057541-47d6bfad-c737-4671-9f4c-9cf95a7f8615.png)
