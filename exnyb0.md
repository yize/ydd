# 排序方式

## 1. 简介[​](#isbU3)
表格排序是按照字段顺序进行排序，所以在每个字段中我们可以选择参照字段
## 2. 操作步骤[​](#P4XkC)
路径：报表页面 >> 选择字段列 >> 字段后面的编辑按钮 >> 数据设置面板 >> 排序
## 3. 参照字段[​](#hLSYo)
选择升序或降序，即可设置参照字段。如下图，虽然商品名称是第一个字段，但可以选择基于创建时间来进行排序以保证创建时间排序在第一优先级![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1682217011228-787cbf77-970b-4cf3-8970-710952dda108.png)
注意：排序不建议使用平均值排序的参考字段不能是公式字段 / 求和字段
