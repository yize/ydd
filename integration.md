# 创建集成&自动化

了解了什么是集成自动化之后，你可以参考本文操作创建集成自动化。
### 从空白创建集成自动化[​](#DtzNg)
你可以参考以下操作创建触发类型为表单事件触发的集成&自动化。登录[宜搭工作台](https://www.aliwork.com/myApp.html)，单击**我的应用**。选择需要创建集成自动化的应用，进入**应用设置**页。单击集成&自动化，依次选择**新建集成&自动化** > **从空白新建**。![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1730185684286-302518c6-e4bf-4ff4-9a04-0bd5813e1c69.png)
在弹窗内设置集成&自动化名称，选择触发类型，单击**确定**完成创建。![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1730186196457-e1b8885c-d9bd-4177-844f-598c6ded52dd.png)
### 从模板创建集成&自动化[​](#cA79r)
你可以参考以下步骤从模板创建集成&自动化流。登录[宜搭工作台](https://www.aliwork.com/myApp.html)，单击**我的应用**。选择需要创建集成自动化的应用，进入**应用设置**页。单击集成&自动化，依次选择**新建集成&自动化** > **从模板新建**。选择需要使用的模版，单击**使用**。![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1731653639432-9a757e1d-1540-43a4-99c9-319507aee2b6.png)
### 常见问题[​](#QagO1)
**Q：可以跨表单复制触发事件吗？**只支持同一个表单内复制触发事件，不支持跨表单复制。![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1653273098762-93bd4a95-5134-4ce6-b2ce-97fbd5818e48.png)
**Q：使用连接器创建1.0待办任务，为什么钉钉待办能正常创建任务，但不显示红点**优先检查钉钉待办是否已设置成无红点，若设置成无红点那钉钉待办不显示红点就是正常情况，修改钉钉待办红点设置如下图所示：**PC端设置**![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1634801967236-14d4632c-7f63-4453-abaa-0fe2230ce210.png)
**移动端设置**
**Q：简易流程在流程节点设置连接器，操作升级高级流程后连接器失效，且复制此表单会报错：应用下包含多个连接器的审批流个数超出限制最多一个**目前高级流程中无连接器组件，故暂不支持连接器功能；复制该流程表单后报错连接器数量超出限制，此情况目前已知，需要等后续产品迭代这块能力，当前可以先取消简易流程的连接器再升级高级流程。![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1636696033869-30c9011e-df53-4191-b1a1-6dbeb68f3ba2.png)
**Q：一个节点有多条业务关联规则怎么排序执行？**业务关联规则目前支持多条规则写入，会按序号进行排序执行，也可通过下述方式进行排序，使用集成自动化的新增、更新、删除等【数据节点】代替，这些节点支持编排顺序，也支持在【运行日志】中查看执行记录。具体设置可以参考以下截图：![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1644910530445-2341c505-0c2c-41f9-ba80-9a538f7570bc.png)
![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1644910556555-fa8e1dcf-5ad0-4160-b710-c00ba9f0d5ce.png)
![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1644910578081-334940a1-3733-42d6-a0ac-95ea683da19c.png)
**Q：对于免登表单，有办法给成员组件传值？**可以通过字段接收人员的userid来实现对目标表成员组件传值。**Q：集成自动化运行结果不符合预期时的排错思路。**首先，检查集成自动化运行日志记录的运行信息；![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1669690545904-e6b688c8-0063-425e-86e2-b343b79cd050.png)
**Q：表单组件配置修改后，自动化中配置的条件操作符失效问题**在集成自动化流中，可配置数据过滤条件，过滤条件中，可针对不同的表单组件，选择不同的操作符；如果表单组件的配置做了修改调整，可能会导致原来配置好的条件操作符失效，需要修改后重新发布；举例：部门组件如果配置的单选模式，条件操作符可选择“等于任意一个”，如果开启了多选，则操作符“等于任意一个”会失效，需要修改为“包含”；![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1716890873117-f9b0cd09-a393-488a-9255-1569afef64ac.png)
![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1716890673432-4c8b3254-7b89-4ecf-997b-07a663e9c5d5.png)
**Q：集成自动化允许自动触发吗？**集成自动化流程支持设置允许自动触发，开启自动触发后，当其他自动化流或流程满足当前自动化流的触发事件时，当前流会被触发。**说明**：自动触发最多支持连续触发三层，即流A => 触发 => 流B，流B => 触发 => 流C，流C => **不能触发** ≠> 流D。
以下情况下，自动触发无法生效：通过集成自动化发起的oa审批，针对oa审批完成后设置的【应用事件触发】不生效，此时手动提交的oa审批可以触发该自动化流。通过集成自动化发起审批，审批对应的流程表单配置的流程【创建】时触发不生效。通过业务关联规则新增、更新数据，目标表单配置的【表单事件触发】不生效。通过集成自动化跨应用触发新增、修改数据，目标表单配置的自动化流不生效。![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1670504173638-1d57d7af-a40b-4a63-a852-bf7dded4fda6.png)
**Q：为什么变更记录不显示集成删除子表的记录？**A：集成自动化变更记录暂不支持查看子表数据变更，支持时间请关注后续产品变化。**Q：通过集成自动化给关联表单组件赋值后，不会触发关联表单的填充？**目前通过集成自动化对表单实例进行更新、新增表单实例都不会触发组件的事件，无论是数据联动、公式计算、onChange事件等，同样也不会触发关联表单的数据填充能力。
