# 报表组件

报表中提供了多种样式的图表，可以通过明细表、数据透视表等查看表单/流程表单数据的明细和汇总；通过柱形、折线、图形、等对数据进行处理，显示出数据的发展趋势、分类对比等结果；通过饼图体现数据中每个部分的比例
| **组件名称** | **用途** |
|---|---|
| [**柱状图**](https://www.yuque.com/yida/support/xybfun) | 使用垂直或水平的柱子显示**类别之间的数值比较** |
| [**饼图**](https://www.yuque.com/yida/support/hb73mq) | 可用于查看每个部门的数据占比情况 |
| [**折线图**](https://www.yuque.com/yida/support/pnsoga) | 用于显示数据在一个**连续的时间间隔或者时间跨度上的变化**，它的特点是反映事物随时间或有序类别而变化的趋势 |
| [**柱线混合图**](https://www.yuque.com/yida/support/wfno1e) | 柱状图，可以直观的看到各组数据差异性，强调个体之间的比较；折线图，可用来分析某类产品随时间变化的销售情况，从而进一步预测未来的销售情况；柱线混合图，则可以同时展示柱状图和折线图 |
| [**仪表盘**](https://www.yuque.com/yida/support/xgetgl) | 可以自行控制组件的大小和随意移动组件位置（靠左、居中、靠右） |
| [**漏斗图**](https://www.yuque.com/yida/support/hd3q1n) | 从漏斗图可以非常直观地看到各个业务的转化程度。从某种意义来说，漏斗图是路径分析的特殊应用，主要是针对关键路径的转化分析 |
| [**词云**](https://www.yuque.com/yida/support/gtyrhp) | 可将表单数据生成一个词云图表 |
| [**雷达图**](https://www.yuque.com/yida/support/grr9lt) | 以从同一点开始的轴上表示的三个或更多个定量变量的二维图表的形式显示多变量数据的图形方法 |
| [**日历热力图**](https://www.yuque.com/yida/support/lyedo6) | 根据日历展示不同时间的进展情况，并根据数值的不同进行配色的调整,**适用于回顾时间周期内代码量、工作情况等场景** |
| [**中国地图**](https://www.yuque.com/yida/support/plcwtt) | 地图行政区划组件，支持中国地图省市县三级，支持中国地图省市县上钻下取 |
| [**基础表格**](https://www.yuque.com/yida/support/stqur9) | 可按Excel表格形式展示出已提交的表单数据 |
| [**交叉透视表**](https://www.yuque.com/yida/support/osbbht) | 日常分析中非常常用的可视化形式之一，类似于Excel。宜搭提供了多级透视，指标计算、合计小计、快捷过滤等功能，还有点击指标直接透出明细数据等方便的操作等待挖掘 |
| [**基础指标卡**](https://www.yuque.com/yida/support/vn761o) | 可以通过【指标】和【图标】的设置，显示关心的指标值与图标展示并支持图标点击下钻 |
| [**文本**](https://www.yuque.com/yida/support/hu9aqr) | 用于展示基础文本信息 |
| [**图片**](https://www.yuque.com/yida/support/pyrhm5) | 用于展示图片信息 |
| [**链接**](https://www.yuque.com/yida/support/tst3xa) | 可用于跳转到 URL 并自由组合参数 |
| [**容器**](https://www.yuque.com/yida/support/qhbihp) | 主要是用于布局，可以与其他组件一起共用 |
| [**分栏**](https://www.yuque.com/yida/support/xvx3lt) | 主要是用于布局，可以与其他组件一起共用 |
| [**选项卡**](https://www.yuque.com/yida/support/pux8db) | 常用的两种形态是普通型与胶囊型；主要是用于布局，可以与其他组件一起共用 |
| [**时间筛选**](https://www.yuque.com/yida/support/hbqowi) | 可以实现【单值】、【区间】的设置且进行筛选 |
| [**下拉筛选**](https://www.yuque.com/yida/support/is9r5k) | 可以实现【单选】、【多选】的设置，并能在输入框中模糊搜索，同时可以设置显示字段以实现用户所见字段和实际查询字段的区分（如显示部门名称，使用部门编码查询）同时还支持【**标签**】模式，实现标签筛选的效果 |
| [**iFrame**](https://www.yuque.com/yida/support/cs2lgv) | 可将其他网页的内容嵌入到当前的设计器页面中 |
