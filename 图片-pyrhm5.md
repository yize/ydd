# 图片

## 1. 简介​

### 1.1 功能简介​

可以用于展示图片信息

亮点：可以自行控制组件的大小，但无法移动图片组件的位置（靠左、居中、靠右）

### 1.2 使用场景​

展示仓库产品数据时需要用配上各产品图片详情。
## 2. 操作步骤​

### 2.1 选择图片组件​

基础 >> 图片 >> 选中后展示在编辑器中，如图：

### 2.2 属性配置​

**图片链接**：复制图片链接并填入，图片会自动展示在该页面上**直接上传**：直接将图片拖拽到虚线框内，支持 jpgjpegpnggifmp 格式文件，图片小于 10M

上传图片方式

放入图片链接**摆放方式：**铺满、适合、拉伸**圆型：**图片四个角的弧度，有大、中、小、无、自定义**圆角大小：**配合着圆型的自定义设置来，若圆型选择自定义，即可设置圆角大小

**title：**图片的基础属性，
不做展示**替代文本：**当图片地址不存在时的保存文本，如图：

替代文本**高级：**唯一标识
