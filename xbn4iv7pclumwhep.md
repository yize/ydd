# 轮播图

## 1. 功能简介[​](#zVIpK)
您可以上传图多张图片并在图片中镶嵌链接，将会以循环播放的方式呈现。
## **2. 功能入口**[​](#PPump)
【选择任意应用】——【编辑门户】——【轮播图组件】![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1688372379706-6211c468-236e-46f2-a40a-f9295b0c9a92.png)
## **3. 设置步骤**[​](#zDk0J)
您可以通过配置以下组件属性来调整最终的组件使用效果。在门户中添加轮播图后，您可以：
### 3.1 调整标题栏[​](#rqbNf)
支持 控制是否“显示标题”。![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1688370440671-c29048f1-1bf2-492f-b3eb-70f5500778b8.png)
### 3.2 配置内容[​](#Ho2fx)
您可以配置轮播图的数据来源，支持2种数据源：动态数据源、静态数据源
| 动态数据源 | 内容来源：选择某应用下的表单 |
|---|---|
| 轮播图片：映射关联底表表单的字段，如轮播图 |
| 链接地址：映射关联底表表单的字段，如链接 |
| 筛选条件：选择某表单后，可设置该表单的数据筛选条件 |
| 显示条数：支持设置“显示条数”，根据“数据条数”实时显示最多可见的轮播图 |
| 静态数据源 | 图片标题：自定义图片标题名称 |
| 图片地址：自定义上传图片 |
| 链接：自定义链接类型，支持打开内部页面 或 打开外部链接 |
