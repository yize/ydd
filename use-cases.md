# 使用案例

本文以县镇村组织的「村集体采购」业务场景为例，介绍如何使用发布应用至上下级组织。
## 前提条件[​](#c0PhN)
在开始之前，你需要先完成以下内容。请确保你已开通上下级组织。详情请参考[上下级组织简介](https://www.yuque.com/yida/support/ampson)。拥有已经开发完成的应用。详情请参考[创建应用](https://www.yuque.com/yida/support/oncnoy)。
## 步骤一：上级组织搭建支持上下级组织的应用[​](#ijgnn)
在应用的接口人设置中，添加接口人配置。详情请参考[接口人管理](https://www.yuque.com/yida/support/ka1xu9)。![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1639359784327-f5023523-5eb0-4123-98f3-c29c66ab906b.png)
针对各接口人，设置各部门的接口人。![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1639360925992-aaf61ea6-847f-4249-8075-4483342798ed.png)
在应用的**流程设计**页面，设置区县审批节点的审批人。![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1639360695063-c003f699-7c7f-4b73-b6b2-9345613cb06e.png)
简易流程设计器介绍：[审批人](https://www.yuque.com/yida/support/brbaos)![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1623979242216-bb897937-b69d-4bc6-ac53-c0888c921201.png)
高级流程设计器介绍：[高级流程设计简介](../../_2/lgrp0w)
## 步骤二：是否允许行业通讯成员访问该应用[​](#rfzTx)
可访问应用的群体分为两类：（1）上下级组织内的成员（2）行业通讯录成员（例如县域行业的村民、教育行业的学生及家长）在应用设置的应用成员设置中，可通过开关控制行业通讯录成员是否可以访问该应用。![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1623979585343-dcf408b4-7ddc-4c22-ac72-d601c23003d2.png)
行业通讯录成员
## 步骤三：安装应用到下级组织[​](#VMUdy)
上级组织将应用分发到下级组织工作台开通了上下级组织即会有分发应用这个按钮，路径：宜搭应用 >> 应用发布 >> 上下级组织应用分发![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1640655900016-b65245b9-6b73-4ac7-8336-d81c3b3d2c18.png)
自动安装到上下级组织工作台如果勾选了自动安装，那么分发应用之后就会自动安装到工作台，没有勾选，在分发应用时会发一个通知确认是否安装到工作台。![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1640655964864-2fd334e8-f189-42b7-b78a-db6716221a92.png)
**说明：**在上下级组织中调用连接器时，需要研发手动加白。
## 步骤四：下级组织被加入上下级组织的成员访问应用[​](#bsfcn)
下级组织在手机钉钉工作台可见此应用，被加入上下级组织的成员或行业通讯录成员，即可访问表单，提交申请；      ![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1620824527217-053f6fc1-2c6c-400b-bd12-ee114ad88ca3.png)
村、镇审批人（接口人）、县审批人，收到审批任务（钉钉通知、宜搭任务中心），进行审批，审批结束，发起人收到审批完成消息通知。（操作同宜搭原有功能）![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1620824795840-f55976aa-7d60-489f-8aa3-388020b65302.png)
![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1620824734154-649364d8-b3e2-4353-99ef-b1a611413d21.png)
[消息通知](../fkfhud)[流程任务中心](../../_2/ea1o8cgypga0lh9t/hg13tb)
## 步骤五：上级组织查看数据[​](#u6Afm)
上级组织可在应用管理后台的「数据管理」中查看所有人员提交的全部数据。![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1640656075376-9a453d6c-190c-48b0-8cd7-b0007b7b6ead.png)
