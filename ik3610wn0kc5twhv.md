# 公告栏

## 1. 功能简介[​](#zVIpK)
您可以快速上传图文摘要，将以公告方式呈现
## **2. 功能入口**[​](#PPump)
【选择任意应用】——【编辑门户】——【公告栏组件】![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1688369076320-558bc418-5163-4dad-bfba-8428668c7e97.png)
## **3. 设置步骤**[​](#zDk0J)
您可以通过配置以下组件属性来调整最终的组件使用效果。在门户中添加公告栏组件后，您可以：
### 3.1 切换组件样式[​](#jtfLU)
点击“切换外观样式”按钮，按需切换组件样式。目前支持以下4种展示方式。![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1688369119078-8386e885-e84e-46aa-af7c-b26b0c122b74.png)
### 3.2 调整标题栏[​](#rqbNf)
支持 控制是否“显示标题”，更改“标题名称”，设置“帮助提示”，控制是否开启“查看更多”配置，开启后需要设置“查看更多”的跳转地址。![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1688369184843-9bce81fb-79ff-43ee-bc26-c01fdf496443.png)
### 3.3 配置内容[​](#Ho2fx)
#### 3.3.1 配置公告数据来源[​](#XlG3C)
您可以配置公告的数据来源，支持2种数据源：动态数据源、静态数据源
| 动态数据源 | 内容来源：选择某应用下的表单 |
|---|---|
| 标题：映射关联底表表单的字段，如名称 |
| 日期：映射关联底表表单的字段，如日期 |
| 链接：映射关联底表表单的字段，如链接 |
| 筛选条件：选择某表单后，可设置该表单的数据筛选条件 |
| 数据条数：支持设置“数据条数”，根据“数据条数”实时显示最多可见的公告数量，如需查看所有清单，可点击“更多”前往查看 |
| 静态数据源 | 标题：自定义标题名称 |
| 日期：自定义日期 |
| 链接：自定义链接类型，支持打开内部页面 或 打开外部链接 |
#### 3.3.2 配置公告配图[​](#pYqtd)
![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1688370023842-15aa47b3-8b69-4ea8-b5e8-95ac97fe5470.png)
