# SaaS应用上架指南

## 前提说明[​](#GYIJw)
请务必通过体验测试组织将SaaS应用自测完毕后再将产品申请上架为商品。
## 应用上架[​](#YXvcp)
支持SaaS品上架到钉钉开放平台>第三方应用统一管理支持SaaS品上架到钉钉应用市场，具备品独立商机、售卖、自运营、服务能力支持SaaS品上架到钉钉酷应用市场，可在群聊内高效协同使用![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1673418476294-f8ee02ca-c4c4-4def-a3bf-9d7a462805bc.png)

| 【4】应用管理>商品上架，填写商品参数规格注意：优先开通顺序：目前暂不支持“优先试用，到期自动转免费” |  |
| 【5】应用管理>商品上架，确定推广模式 |  |
标准SaaS应用服务产品在钉钉上架发布后，主要分为以下两种推广形式：**应用市场推广**：如果你已有应用产品或计划开发应用，只要你有清晰的商业模式，且符合钉钉应用广场上架标准，通过审核后即可上架到应用广场推广。应用上架后，客户即可在钉钉应用广场搜索、查看和安装及购买应用商品。**自有渠道推广**：如果你已有应用产品或计划开发应用，可通过钉钉提供的接口生成线下推广二维码，在服务商自有渠道或钉钉端外渠道（例如外投宣传广告、第三方官网等）推广应用商品。客户可以通过扫码安装或点击链接安装应用。
### 2 上架到钉钉酷应用市场[​](#nLBn9)
加入钉钉合作伙伴组织，在OA审批中搜索“[酷应用上架申请](dingtalk://dingtalkclient/action/openapp?app_id=-4&container_type=work_platform&corpid=ding77a7af8924a84af0&ddtab=true&redirect_type=jump&redirect_url=https://aflow.dingtalk.com/dingtalk/mobile/homepage.htm?back_control=false&backcontrol=false&corpid=ding77a7af8924a84af0&dd_progress=false&dd_share=false&ddtab=true&showmenu=false#/custom?pcredirect=self&processCode=PROC-21027998-89B9-4D56-862D-4122960CA038)”，按照表单要求填写相关物料信息，提交审批待钉钉开放平台运营人员审批，审批通过后，即可在酷应用市场看到已上架的应用。![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1673935886084-476854cf-ac8c-413b-bead-d4f630002e7b.png)
