# 流程预览

宜搭审批流程支持在提交时预览流程。此功能允许流程发起者在提交前预览整个审批流程，通过简便的勾选项激活系统自动推送审批通知至审批人钉钉个人聊天的机制，从而在流程启动阶段便高效化解潜在障碍，全面提升审批流程的顺畅度与用户体验。流程打印：打印模板支持配置只打印最后一次审批记录。
## 审批预览[​](#I9C0t)
流程发起时，系统自动以弹窗的形式进行**流程预览，**将审批流程中发起人节点、审批人节点及执行人节点涉及人员进行展示。![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1718353194166-80b7d511-f40f-49f0-b7e3-ce9cf8b43088.png)
**注意：**抄送人节点并不会展示在本流程预览窗口中。聊天消息卡片当前仅能发送给当前流程距离提交人节点最近的审批人。
## 消息提醒[​](#REJ83)
勾选**「通过聊天发送给当前审批人」**后点击**确认**，则会推送相应审批卡片到聊天窗口。且卡片内的流程状态以「印章」的形式进行展示。![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1718353398814-ec49bbad-8851-4ac5-8bc7-32e908f7f6c2.png)
| 流程中 | 流程拒绝 | 流程撤销 |
|---|---|---|
|  |  |  |
点击卡片内「查看详情」，即可跳转到流程表单页面进行操作。![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1648198474035-6f0e9e49-c6d9-4894-827a-0a2c957f4f68.png)
当前，流程表单-提交数据-预览数据、数据详情页-审批进度、流程测试-审批流程等均支持查看完整流程走向。用户可以查看整个流程的走向和结构，展示各个流程走向和节点关系，通过完整的拓扑图了解各个节点之间的关系。请参考[流程支持完整走向图](https://www.yuque.com/yida/test/uzcgrfgcozbfzchi)。
