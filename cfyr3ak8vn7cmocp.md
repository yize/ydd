# 邮件节点

邮件通知用于在流程执行到某个节点时，通过已经配置完成的邮件模板，向指定人员发送邮件通知。你可以参考本文操作了解如何配置发送邮件节点。
## 准备工作[​](#o6XxQ)
在使用审批流程发送邮件通知功能之前，你需要先添加邮箱帐号，详情请参考[邮箱管理](https://www.yuque.com/yida/support/dowbnvn7aasg1z07)。
## 添加发送邮件节点[​](#p1yIu)
登录[宜搭工作台](https://www.aliwork.com/myApp.html)，单击**我的应用**。选择需要的应用，进入**应用设置**页。单击**集成&自动化**，选择已经创建的集成自动化流，单击**编辑**。将光标移动至流程节点的连接线位置，会弹出`**+**`按钮，单击`**+**`，选择**消息节点** > **发送邮件**节点。![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1733713618076-bc393492-dca5-461b-bb58-3d9cdf4a5dbb.png)
## 配置发送邮件[​](#UsfyG)
你可以参考以下步骤配置发送邮件节点。选择**发送邮件**节点，在右侧节点配置弹窗中**设置邮箱地址**。**发件人邮箱帐号**：配置发件人的邮箱帐号，即以谁的名义发送邮件。**说明**：如果你还没有添加邮箱帐号，可以单击右侧**帐号管理**添加邮箱，详情可参考[邮箱管理](https://www.yuque.com/yida/support/dowbnvn7aasg1z07)。
**收件人**：该邮件通知的收件人，支持设置超送人和密送人。**说明**：收件人支持使用表单内字段。
![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1733713667784-db3de028-5faa-4986-8443-88c95084a6fb.png)
设置邮件内容，邮件内容支持自定义和消息模板两种模式。**自定义**：你可以直接设置邮件主题与邮件内容，邮件内容支持富文本。**使用通知模板**：使用通知模板是复用你在**宜搭平台管理**页内配置消息通知模板，在此处修改模板内容不会影响源模板。如何配置消息通知模板，可参考[消息通知模板](https://www.yuque.com/yida/support/fkfhud)。**说明**：支持使用表单内字段进行拼接。
![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1733714070851-8e1e8658-bff3-4f23-ba7c-bbbffae78fa3.png)
配置完成后，单击**保存**。
## 邮件支持组件类型说明[​](#ojre4)
下为各个组件支持/不支持批量导出数据的组件类型说明：
| **组件类型** | **组件名称** | **是否支持邮件引用** | **邮件引用数据格式** |
|---|---|---|---|
| **常用组件** | 单行文本 | 支持 | 文本格式 |
| 多行文本 | 支持 | 文本格式 |
| 数值 | 支持 | 数字格式 |
| 评分 | 支持 | 数字格式 |
| 单选 | 支持 | 文本格式 |
| 复选 | 支持 | 文本格式。导出所有选中的选项，多选项英文逗号隔开 |
| 日期区间 | 支持 | 日期格式 |
| 日期 | 支持 | 日期格式 |
| 成员 | 支持 | 1.不☑️显示成员编号，则显示成员姓名2.☑️显示成员编号，显示成员姓名+成员编号 |
| 部门 | 支持 | 1.不☑️是否展示部门全路径，仅显示部门名称2.☑️是否展示部门全路径，显示部门完整路径 |
| 图片上传 | 支持 | 图片路径，登录钉钉后才可访问 |
| 附件 | 支持 | 图片路径，登录钉钉后才可访问 |
| word模版 | 支持 | 图片路径，登录钉钉后才可访问 |
| 手写签名 | 支持 | 图片路径，登录钉钉后才可访问 |
| 智能发票 | 支持 | 图片路径，登录钉钉后才可访问 |
| 子表单 | 支持 | 文本格式 |
| **高级组件** | 下拉单选 | 支持 | 选中的选项 |
| 下拉复选 | 支持 | 导出所有选中的选项，不同选项间需用英文逗号隔开 |
| 级联选择 | 支持 | 文本格式 |
| 国家/地区 | 支持 | 文本格式 |
| 地址 | 支持 | 文本格式 |
| 定位 | 支持 | 文本格式 |
| 关联表单 | 支持 | 文本格式，字段+实例ID |
| 流水号 | 支持 | 文本格式 |
| 企业 | 支持 | 文本格式 |
| 银联支付 | ❌不支持 | / |
| 预算费控 | ❌不支持 | / |
| 分组 | ❌不支持 | / |
| 关联查询 | ❌不支持 | / |
| 富文本 | ❌不支持 | / |
| 图文展示 | ❌不支持 | / |
| **常用组件** | 身份证号 | ❌不支持 | 出于隐私保护考虑不支持邮件导出 |
| 电话号码 | ❌不支持 | 出于隐私保护考虑不支持邮件导出 |
