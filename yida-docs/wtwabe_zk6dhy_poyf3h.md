功能简介​宜搭与e签宝打通升级，提供电子签章能力，才得以实现文件/合同随时随地签署，且具有法律效益。1.2 使用场景​以往合同签署、保证书签订、约定签署等具有法律效益的工作，通常在线下以纸质合同形式进行，但传统纸质合同存在保存难、签署时间长，传递久等问题，如今使用宜搭电子签章功能即可解决。1.3 不支持功能​电子签章目前以自定义组件形式接入，当前不支持以下功能：不支持 Excel 批量导入/导出/修改/发起/编辑；不支持数据管理页排序、筛选、列表展示；不支持集成自动化使用；旧版高级/简易流程设计器使用电子签章；浏览器发起签署，需在钉钉 PC 端；2. 使用指南​2.1 开通e签宝​e签宝使用前提：仅支持钉钉组织管理员身份开通e签宝；企业实名需钉钉组织管理员操作。2.1.1 开通路径​使用电子签章组件需要先在钉钉组织中安装e签宝功能，开通方式如下：路径一：手机钉钉 >> 工作台 >> 应用中心搜索 e签宝 >> 免费开通路径二：手机钉钉 >> 消息页面右上角+号 >> 扫一扫 >> 扫描下方二维码 >> 选择所属团队 >> 免费开通路径三：PC 端钉钉 >> 工作台 >> 应用中心搜索 e签宝 >> 免费开通 若已经开通e签宝，在此页面直接选择我已开通e签宝。2.1.2 授权​开通e签宝后，需要进行授权才可在宜搭使用。授权路径：进入宜搭 >> 平台管理 >> 基本信息 >> 高级功能 >> E 签宝 >> 去授权 ，进入e签宝页面，进行授权操作；授权后，e签宝页面点击返回可回到平台管理，自动进入第三步。2.1.3 企业实名​钉钉组织安装e签宝后，需要在 e签宝进行企业实名认证后，才可发起签署和签署，仅支持钉钉管理员可通过以下方式进行企业实名认证。手机端实名路径：手机钉钉 >> 工作台 >> e签宝 >> 我的 >> 企业未实名 >> 点击【完善企业信息】注：企业实名认证前，企业经办人需要先完成实名，如果经办人未完成实名，会先引导经办人完成个人实名，具体操作步骤可以参考【个人实名】详情认证步骤请查看文档：e签宝如何进行企业实名认证电脑端实名路径：电脑钉钉 >> 工作台 >> e签宝 >> 点击【待完善企业信息】进行企业实名认证。详情认证步骤请查看文档：e签宝如何进行企业实名认证企业实名流程当中，需要核验填写的企业信息真实性，为了满足合规要求，需要办理人授权同意平台方采集必要信息。2.1.4 充值使用​e签宝会免费赠送5份合同，具体获取方式可参考文档：如何免费领取5份电子合同。e签宝套餐仅支持手机端进行购买，须联系钉钉管理员进行订购下单。路径一：手机钉钉 >> 工作台 >> e签宝 签署人​签署人须保持至少有一个。签署人类型内部人员：钉钉组织架构内的人员；外部人员：钉钉组织架构外的人员。签章类型企业章：需配置签署企业名称；个人章：需进行个人实名认证。支持同时勾选以上 2 种签章类型。签署人内部人员：具体签署的人员无法直接设置固定值，可选择字段类型包括当前表单的人员单选、人员多选，支持子表字段。请确保输入签署人真实姓名，否则将会导致签署人后续无法完成实名签署。外部人员签署人：可选择字段类型包括当前表单的单行文本、单选、下拉单选、关联表单，支持子表字段；签署人手机号/邮箱：应确保本字段填写数据为签署人本人在用，签署人将会通过短信或者邮箱收到签署链接，可选择字段类型包括当前表单的单行文本、单选、下拉单选、关联表单，支持子表字段；签署企业名称：签章类型为企业章时，需配置签署企业名称，可选择字段类型与签署人姓名一致，可选择字段类型包括当前表单的单行文本、单选、下拉单选、关联表单，支持子表字段,。2.2.3 添加抄送人​抄送人类型内部人员：钉钉组织架构内的人员；外部人员：钉钉组织架构外的人员。抄送主体抄送主体分为个体、企业 2 种。当抄送主体为个体，该签署文件归属于个人，可直接下载；当抄送主体为企业，会校验是否拥有企业文件下载权限，如无操作权限需下载，请联系e签宝管理员。注：抄送主体不支持多选。抄送人内部人员，仅支持选择表单中的成员组件；外部人员，抄送人栏可选择表单中的文本组件，且需要必填抄送人手机号或者邮箱；请确保输入抄送人真实姓名，否则将会导致抄送人后续无法查看。2.3 发起签署​2.3.1 如何发起签署​支持通过以下 2 个路径发起签署/签署文件。路径一：表单设计 >> 添加组件库添加电子签章 >> 访问提交表单 >> 发起签署路径二：进入表单详情数据 >> 打印 >> 选择打印模板 >> 签署文件注：表单设计中不添加电子签章组件也可在打印预览页面进行签署文件。2.3.2 e签宝-发起合同任务​在宜搭应用内发起签署后会跳转到 e签宝-发起合同任务页面，需填写以下信息内容。更多详情可查看文档：如何发起一份合同任务。（1） 任务基本信息​任务主题默认格式：XXX发起的流程，支持自定义其他内容。注：任务主题签署完成后不支持二次修改；最多支持 50 个汉字。（2）合同文件​支持格式：pdf、doc、docx、png、jpg、jpeg、xlsx、xls、jfif ；签署文件 最多 50 份 ，附属文件最多 50 份；附属文件只供对方查阅，不用于签署；默认展示表单附件中文件或打印文件。（3）任务参与人​顺序设置签署顺序设置默认关闭，需要手动开启，开启后就可以对填写和签署顺序进行设置，上下拖拽即可。参与人参与人员：直接选择企业内或输入企业外的人员姓名，若输入企业外的人员姓名，需补充对应人员的手机号/邮箱；印章类型：个人章、企业章、法定代表人章（勾选法定代表人印章，需要在企业控制台已经申请过法人印章，不然无法签署），支持多选；签署要求：手绘签名、个人印章、AI 手绘章，支持多选；添加同事签：组织内的成员签署（同属于一个钉钉组织内）点击添加同事签，选择钉钉成员信息；添加参与人：组织外的成员签署（不在同一个钉钉组织内）点击添加参与人，填写参与方的姓名和手机号码；注：请确保输入签署人真实姓名，否则将会导致签署人后续无法完成实名签署。抄送人抄送人员：直接选择企业内或输入企业外的人员姓名，若输入企业外的人员姓名，需补充对应人员的手机号/邮箱；抄送主体分个人和企业。（4）其他设置​签署截止日期默认选择不设置签署截止日期，若设置了截止日期，则所有签署方必须在截止日期前完成签署，文件到期后将无法签署，需要重新发起。注：宜搭目前无法返回过期状态，故签署截止日期只能在e签宝内设置。文件到期提醒设置默认选择不设置文件到期提醒，若设置文件到期日后，届时会发送通知提醒进行合同续签事宜。2.3.3 查看签署合同​（1）查看入口​PC 端钉钉 >> 工作台 >> e签宝 >> 最近文件 >> 查看全部 >> 我发起的/我收到的/抄送我的 （2）签署状态​目前有以下 6 种签署状态，如下图：2.4 签署打印​电子签章组件默认参与打印，系统会自动展示业务 ID、签署状态 code、签署状态 3 个字段。注意：不支持批量发起签署打印；支持单条打印流程；3. 常见问题​3.1 为什么提示签署文件为空，请上传后再试？​请检查表单中附件组件是否有上传有文件，若未上传则会出现以下报错提示。3.2 外部人员会收费吗？​不会，会以发起人按照签署完成的合同份数付费。e签宝会免费赠送5份电子合同，具体获取方式可参考文档：如何免费领取5份电子合同。3.3 e签宝、宜搭、钉钉之间的关系？​钉钉使用 e签宝在钉钉安装e签宝，企业实名认证后创建企业空间，钉钉组织成员即e签宝企业人员。默认拥有一个企业章，还可额外创建企业章、法人章，用于企业盖章签署。支持印章、合同等权限管理。宜搭使用e签宝首先需在钉钉组织中安装e签宝，授权宜搭连接钉钉e签宝，可发起签署，请前往钉钉e签宝管理印章和权限。非钉钉企业使用e签宝签署：打开短信签署地址，手机号验证登录，盖章签署（需实名认证）。发起签署：手机号登录e签宝平台，实名认证后发起签署。印章管理：在e签宝平台，企业实名认证后创建企业空间，默认拥有一个企业章，还可额外创建企业章、法人章，用于企业盖章签署。签署人可以不使用宜搭、钉钉，只需手机号验证登录，即可盖章签署合同（首次需实名认证）3.4 为什么在浏览器发起签署一直提示请在新页面e签宝中发起签署？​e签宝支持在 PC 端发起签署，若在浏览器中发起则会提示：请在新页面e签宝中发起签署。3.5 为什么点击去授权无反应？​请管理员在钉钉管理后台 >> 工作台 查看是否有2个宜搭应用，停用较老的应用后再进入宜搭尝试操作授权。3.6 签署组件同步文件组件不能与签署文件组件相同​请检查表单中的附件组件是否已作为签署文件，签署文件与签署完成同步文件不能使用同一个附件组件。3.7
