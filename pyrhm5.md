# 图片

## 1. 简介[​](#qGHe0)
### 1.1 功能简介[​](#GkM2i)
可以用于展示图片信息![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/jpeg/1625964989401-9a31a471-befa-495c-8128-90058e44d85b.jpeg)
亮点：可以自行控制组件的大小，但无法移动图片组件的位置（靠左、居中、靠右）![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1639987048060-aced159e-e01c-425e-8493-dcdca5461d8b.png)
### 1.2 使用场景[​](#miDPf)
展示仓库产品数据时需要用配上各产品图片详情。
## 2. 操作步骤[​](#cbZfC)
### 2.1 选择图片组件[​](#uxVjv)
基础 >> 图片 >> 选中后展示在编辑器中，如图：![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1639989786635-001cdd63-d2ad-4e89-a7f9-897e737625af.png)
报表页面编辑
### 2.2 属性配置[​](#BIH4i)
**图片链接**：复制图片链接并填入，图片会自动展示在该页面上**直接上传**：直接将图片拖拽到虚线框内，支持 jpgjpegpnggifmp 格式文件，图片小于 10M![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1639989826805-37997360-d818-434a-bc4e-160f418d9e1a.png)
上传图片方式![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1639990186573-44ea32f2-ed7c-4fa8-976a-9f4e541c64a5.png)
放入图片链接**摆放方式：**铺满、适合、拉伸**圆型：**图片四个角的弧度，有大、中、小、无、自定义**圆角大小：**配合着圆型的自定义设置来，若圆型选择自定义，即可设置圆角大小![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1639990239455-8962c80c-9c1d-43fa-99e0-bb0bc38f51fd.png)
**title：**图片的基础属性，不做展示**替代文本：**当图片地址不存在时的保存文本，如图：![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1639990302984-111161ee-a30e-4195-9add-e67da67a80bf.png)
替代文本**高级：**唯一标识
### 2.3 配置图片样式[​](#JvlAh)
