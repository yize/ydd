# 从 Excel 创建应用

宜搭低代码应用支持五种创建方式，本文介绍如何使用 Excel 创建应用。
## 操作步骤[​](#RTsAx)
登录宜搭[工作台](https://www.aliwork.com/workPlatform)。单击右上角**创建应用**，然后在弹出的对话框中单击**从 Excel 创建应用**。![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1704882680991-f3efcfbf-6b33-4d1c-99ee-56f39c5ed865.png)
在**从 Excel 创建应用**页面中，**选择 Excel **文件。
你可以选择**从 Excel模板中选择**或**从本地上传 **Excel 文件。![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1704884850005-2b6dcef3-0cc8-429f-bf68-870ae47d7e20.png)
在字段设置步骤，参考以下信息配置表单字段，然后单击**下一步**。选���需要修改的字段，单击字段后的 `**▽**`** **按钮。设置该字段**是否必填**。设置**字段标题**。单击字段类型下拉选，选择该字段类型。
宜搭会自动识别 Excel 的字段类型，如：文本输入框、数字输入框、日期输入框、单选、成员选择框等。![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1704885468180-c701b5b3-4abb-4132-9ce0-9482f8fe2f07.png)
在**流程设计**步骤，选择是否开启审批流程。**说明**：当新增数据需要审批时，即可开启流程，完成流程图设计。流程设计步骤可参考[流程设计](https://www.yuque.com/yida/support/oihk8g)。通过 Excel 创建的应用流程设计审批人目前最多只能添加三位。
在**应用设置**步骤，配置应用名称和权限，最后单击**创建应用**完成应用创建。![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1704885873961-d4b816ea-b6c3-44d8-a6cd-c0019f732707.png)
