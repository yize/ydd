# AI填表

## 1、简介[​](#wBcbQ)
### 1.1功能简介[​](#U22sB)
AI填表组件是一种基于人工智能技术的自动化工具，允许用户通过下达简单的书写指令来自动填写表单字段，并能够提交和保存表单。其主要特点包括：**智能识别**：AI可以理解并解析用户的自然语言指令，准确识别需要填写的字段。**自动化填写**：根据用户提供的指令或预设规则，AI自动填充表单中的各个字段。**实时校验**：在填写过程中，AI会实时检查数据的正确性和完整性，确保表单符合要求。**一键提交**：完成填写后，用户只需点击一次按钮即可提交表单，并进行保存。
### 1.2应用场景[​](#OCqr2)
**人力资源管理**：HR部门可以通过AI填表组件快速处理大量的入职申请、离职登记、员工信息更新等表单，减少人工输入错误，提高工作效率。**客户信息录入**：销售和服务团队可以利用AI快速录入客户的基本信息、联系方式、购买记录等，提升客户服务质量。**财务报销**：员工可以使用AI自动生成报销申请表，填写费用明细、发票信息等内容，简化报销流程，加快审批速度。**医疗记录**：医生和护士可以利用AI快速填写患者病历、诊断记录等，提高诊疗效率，减少人为失误。
## 2、操作步骤[​](#QYKg6)
### 2.1 入口[​](#ijSQn)
进入表单页面，选择整体表单容器，右侧属性栏- AI表单配置，在右侧进行组件配置。
### 2.2 配置组件参数[​](#mCpD0)
| **图例** | **组件名称** | **基本说明** |
|---|---|---|
|  | 是否启用 | 默认不启用，启用后将在页面中展示如下按钮 |
| 系统提示语 | 请输入系统提示语，使用 ｛fieldld｝｝引用表单字段 |
| AI模型 | 默认通义千问，可下拉选择 |
| 指定填写字段 | 可选择指定填写该表中的某一字段 |
| 允许用户输入额外的提示词 | 允许用户输入自定义提示语，会在弹层中允许输入 |
| 按钮设置-按钮文字 | 可输入按钮文字 |
| 按钮设置-按钮类型 | 主按钮/默认按钮/虚线按钮/链接按钮/文本按钮 |
| 按钮设置-按钮描述 | 鼠标悬停时显示的文字 |
