# 智能公式

进入宜搭，进入某个应用的某个表单，点击「字段默认值-公式」入口，打开公式面板，即可通过AI生成公式![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1692860087830-717e330f-3674-410d-8695-766ce013a3b2.png)
## **支持场景**[​](#NZu08)
智能公式可以辅助应用开发，准确命中公式诉求，自动补齐公式参数，低门槛实现高频场景的公式配置 。基于自然语言推荐公式 ，生成101个高频公式，并完成字段拼接支持的公式生成类型包括 文本、时间、逻辑、数学、集合、人员、校验等
## 使用案例[​](#j2fSQ)
| **步骤** | **描述** | **示意图** |
|---|---|---|
| 01 | 选定需要公式生成的字段，输入prompt |  |
| 02 | 等待公式生成，确认是否符合预期 |  |
| 03 | 确认合理，点击【完成】插入公式 |  |
## 注意事项[​](#b6CMO)
由于公式需要基于字段进行计算，请在指令中指明对哪几个字段进行处理，输入的字段名称越完整越好目前单次生成为单公式，暂不支持嵌套公式，因此对于复杂嵌套公式，可以逐步生成，辅助配置
