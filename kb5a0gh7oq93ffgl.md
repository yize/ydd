# 快捷入口

## 1. 功能简介[​](#zVIpK)
您可以关联应用内页面或引用外部链接，满足不同场景下快速查看页面的诉求。
## **2. 功能入口**[​](#PPump)
【选择任意应用】——【编辑门户】——【快捷入口组件】![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1688356179553-a0af148f-c602-4aa4-8fc0-72eb8882bb70.png)
## **3. 设置步骤**[​](#zDk0J)
您可以通过配置以下组件属性来调整最终的组件使用效果。在门户中添加快捷入口组件后，您可以：
### 3.1 切换组件样式[​](#jtfLU)
点击“切换外观样式”按钮，按需切换组件样式。目前支持以下7种展示方式。![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1688356378417-90ef52ba-6303-4c80-b7a7-26eddbbc69c8.png)
### 3.2 调整标题栏[​](#rqbNf)
支持 控制是否“显示标题”，更改“标题名称”，设置“帮助提示”，控制是否开启“查看更多”配置，开启后需要设置“查看更多”的跳转地址。![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1688356582201-b7b0d0bd-805a-4d4f-bb11-f7ace2f745bf.png)
### 3.3 配置内容[​](#Ho2fx)
您可以通过快速新建一个快捷入口，也可以批量编辑快捷入口信息，包括下述内容：
|  | 设置图标 |
|---|---|
| 入口名称 |
| 辅助描述：当组件切换不同的样式后，如下述效果，可自定义“辅助描述文案”展示相关信息 |
| 归属分组：填写该入口计划放到某分组下的分组名称 |
| 入口类型支持打开内部页面，选择应用列表内的某页面支持打开外部链接，自行输入跳转的外部地址 |
![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1688356881343-a25a5bb7-c91a-465a-9043-f5833415a176.png)
“新增单个快捷入口”效果![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1688356858545-13843017-4781-44ea-9d1d-5a0a9b41c341.png)
“批量编辑”快捷入口效果
### 3.4 设置分组[​](#sYukX)
当有多个快捷入口且已编写“所属分组”名称，可以控制是否“开启分组”，开启后可选择分组展示样式![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1688356754998-10651f36-e439-43b3-b6f2-ca94a0e8103c.png)
“选项卡”效果![](https://yida-support.oss-cn-shanghai.aliyuncs.com/static/png/1688356764438-3a06c3f1-a6a8-4464-9179-54d7532a4750.png)
“平铺”效果
