# 从 Excel 创建表单

未升级到新版信息架构的组织，请 **点此查看**使用手册
## 1. 简介​

### 1.1 功能简介​

通过 Excel 创建表单可以快速、便捷的完成表单设计，并且可以将 Excel 中的数据直接导入进表单中，进行数据管理。
### 1.2 使用场景​

在进销存的场景中，需要录入产品名称、规格、数量以及单价金额时，可以通过 Excel 表格录入好组件名称和数据，从而快速的导入产品信息数据底表
## 2. 操作步骤​

### 2.1 选择 Excel 表​

路径：新建表单 >> 从 Excel 导入 >> 选择 Excel 表

### 2.2 数据预览​

可在线预览本地 Excel 表格的数据是否有误或是否选择正确

### 2.3 表单设置​

设置当前创建的表单名称以及组件名称和组件类型

### 2.4 导入数据​

上述步骤完成后，点击表单设置页面右下角的导入即可进行导入数据

## 3. 导入规则​

上传的 Excel 表符合以下规范：
• 不能存在合并的单元格
• 文件大小不超过 2MB
• 仅支持 (*.xls 和 *.xlsx) 文件
• 请确保你需要导入的sheet表头中不包含空的单元格，否则该 sheet 页数据系统将不做导入
